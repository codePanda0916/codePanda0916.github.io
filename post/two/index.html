Array属性及方法

Oct 24, 2019
6 min read

JavaScript的Array对象是用于构造数组的全局对象，数组是类似于列表的高阶对象。

第一部分：属性

1.1 Array.length

length是Array的实例属性。返回或设置一个数组中的元素个数，该值是一个无符号32-bit整数，并且总是大于数组最高项的下标。length的属性值是一个0到2^32 - 1的整数； 常使用于遍历数组和截断数组；

1.2 Array.prototype

Array.prototype表示Array构造函数的原型，并允许你向所有Array对象添加新的属性和方法。 Array的实例继承自Array.prototype

  Array.prototype.length // 0

第二部分：方法

2.1 Array.from()

从一个类似数组或可迭代对象创建一个新的、浅拷贝的数组实例。 语法： Array.from(arrayLike, mapFn, thisArg)

var my = Array.from('foo');
console.log(my);
// ["f", "o", "o"]

var my = Array.from([1,2,3], x => x+x);
console.log(my);
// [2, 4, 6]

2.2 Array.isArray()

Array.isArray()用于确定传递的值是否是一个Array;

Array.isArray([1,2]);
// true
Array.isArray('foo');
// false
Array.isArray({foo: 123});
// false
Array.isArray(undefined);
// false

2.3 Array.of()

Array.of()方法创建一个具有可变参数的新数组实例，而不考虑参数的数量或者类型。

Array.of(7)     //  [7]
Array.of(1,2,3) //  [1,2,3]
Array.of([1,2,3]) // [[1,2,3]]

Array(0)     // []
Array(1,2,3) // [1,2,3]
Array(7)     // [,,,,,,] or [empty x 7]

会改变自身的方法

2.4 Array.copyWithin(taget, start, end)

target: 必需，从该位置开始替换数据； start: 可选，从该位置开始读取数据，默认为0，如果为负数，从倒数开始 end: 可选，到该位置前停止读取数据，默认为数组长度，如果为负数，表示倒数

['1','22','333','4444','55555'].copyWithin(0,3,4);
// ["4444", "22", "333", "4444", "55555"]
['1','22','333','4444','55555'].copyWithin(0,3);
// ["4444", "55555", "333", "4444", "55555"]

2.5 Array.fill(value,start,end)

value: 用来填充数组的值； start: 起始索引，默认为0； end: 终止索引，默认值为Array.length;

[1,2,3,4].fill('222');
// ["222", "222", "222", "222"]
[1,2,3,4].fill('222', 1);
// [1, "222", "222", "222"]
[1,2,3,4].fill('222',2,3);
// [1, 2, "222", 4]

2.6 Array.pop()

从数组中删除最后一个元素，并返回该元素的值。更改数组的长度；

var array = [1,2,3,4];
var newArray = array.pop();
console.log(newArray);
// 4
console.log(array);
// [1, 2, 3]
[].pop()
// undefined

2.7 Array.push()

push方法将一个或者多个元素添加到数组的末尾，并返回新数组的长度。

var array = [1,2,3,4];
var newArray = array.push(3);
console.log(newArray);
// 3
console.log(array);
// [1, 2, 3, 4, 3]

2.8 Array.reverse()

Array.reverse()将数组中的元素的位置颠倒，并返回该数组；

var arr= [1,2,3,4,5];
var newArr = arr.reverse();
console.log(newArr);
// [5, 4, 3, 2, 1]
console.log(arr);
// [5, 4, 3, 2, 1]

2.9 Array.shift()

从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度。

var arr= [1,2,3,4,5];
var newArr = arr.shift();
console.log(arr);
// [2, 3, 4, 5]
console.log(newArr);
// 1

2.10 Array.sort()

sort()方法用原地算法对数组的元素进行排序，并返回数组。默认排序顺序是在将元素转换为字符串，然后比较它们的UTF-16代码单元值序列时构建的。

var array = [20,18,59,90,62,39,58,42,77];
array.sort();
// [18, 20, 39, 42, 58, 59, 62, 77, 90]
array.sort(function(a,b){
  return a-b;
})
// [18, 20, 39, 42, 58, 59, 62, 77, 90]
array.sort(function(a,b){
  return b-a;
});
// [90, 77, 62, 59, 58, 42, 39, 20, 18]

2.11 Array.unshift()

unshift()方法将一个或者多个元素添加到数组的开头，并返回该数组的新长度；

var array = [1,2,3];
array.unshift(11,22,33);
// 返回新数组的长度 6
console.log(array);
// [11, 22, 33, 1, 2, 3]

2.12 Array.splice(start, deleteItem, item1…)

splice可实现删除、修改(or替换)，添加的功能； 参数说明： start: 修改的开始位置; deleteItem: 整数，表示要移除数组元素的个数； item1: 要添加进数组的元素；

// 删除
var array = ['111','222','333','444','555','666'];
array.splice(2, 3); // ["333", "444", "555"]
console.log(array); // ["111", "222", "666"]

// 修改
var array = ['111','222','333','444','555','666'];
array.splice(2, 1 ,9999); // ["333"]
console.log(array); // ["111", "222", 9999, "444", "555", "666"]

// 添加
var array = ['111','222','333','444','555','666'];
array.splice(2, 0 ,9999, 199); // []
console.log(array); // ["111", "222", 9999,199, "333", "444", "555", "666"]

不会改变自身的方法

2.13 Array.concat()

concat()用于合并两个或多个数组，此方法不会更改现有数组，而是返回一个新数组； 语法： var newArray = oldArray.concat(val1,val2,…, valN);

var alpha = ['a', 'b', 'c'];
var oldArray = [1,2,3];
oldArray.concat(alpha); // [1, 2, 3, "a", "b", "c"]

var num1 = [1,2,3];
var num2 = [4,5,6];
var num3 = [7,8,9];
var nums = num1.concat(num2, num3);
console.log(nums); // [1, 2, 3, 4, 5, 6, 7, 8, 9]

2.14 Array.includes()

includes()方法用来判断一个数组是否包含一个指定的值，根据情况，如果包含则返回true，否则返回false； 语法：arr.includes(valueToFind, fromIndex); valueToFind: 需要查找的元素值， fromIndex：可选，从索引fromIndex开始搜，默认为0； // 注意：对象数组不能使用includes方法来检测；

var Array1 = [1,2,3,4];
console.log(Array1.includes(2)); // true

var pets = ['cat', 'dog', 'cats'];
console.log(pets.includes('Cats')); // false

2.15 Array.join()

join()方法将一个数组的所有元素连成一个字符串并返回这个字符串。如果数组只有一个数组，那么将返回该项目而不使用分隔符； 语法：arr.join(separator); separator: 可选，缺省则默认用逗号“，”分割，如果separator是空字符串(“”),则所有元素之间都没有任何字符；

var elements = ['Fire', 'Air', 'Water'];
console.log(elements.join()); // Fire,Air,Water
console.log(elements.join("")); // FireAirWater
console.log(elements.join("-")); // Fire-Air-Water

2.16 Array.slice()

slice()方法返回一个新的数组对象，这一对象是一个由begin和end决定的原数组的浅拷贝(begin, end)。原始数组不会被改变。 语法： arr.slice(begin, end); begin: 可选，从该索引开始提取数组元素； end: 可选，提取终止处的索引；从该索引处结束提取原数组元素；

var oldArray = ['ant', 'bison', 'camel', 'duck', 'elephant'];
console.log(oldArray.slice(2)); //  ["camel", "duck", "elephant"]
console.log(oldArray.slice(2,3)); // ["camel"]
console.log(oldArray.slice()); //["ant", "bison", "camel", "duck", "elephant"]

2.17 Array.toString()

toString()返回一个字符串，表示指定的数组及其元素；

var newArray = ['ac','12','rr'];
console.log(newArray.toString()); // "ac,12,rr"
var newArray = [11,22,333];
console.log(newArray.toString()); // "11,22,333"

2.18 Array.toLocaleString()

toLocaleString()返回一个字符串表示数组中的元素。数组中的元素将使用各自的toLocaleString方法转成字符串，这些字符串将使用一个特定语言环境的字符串(例如一个逗号”,“)隔开。 语法：arr.toLocaleString(locales, options); locales: 可选，带有BCP 47语言标记的字符串或者字符串数组； options: 一个可配置属性的对象，对于数字Number.prototype.toLocaleString(),对于日期Date.prototype.toLocaleString();

var array = [1, 'a', new Date()];
console.log(array.toLocaleString()); // "1,a,2019/10/26 下午8:35:07"
console.log(array.toLocaleString('en', {timeZone: "UTC"})); // 1,a,10/26/2019, 12:36:01 PM

2.19 Array.indexOf()

indexOf()方法返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1； 语法：arr.indexOf(searchElement, fromIndex); searchElement: 要查找的元素 fromIndex：可选。从索引fromIndex开始查找，默认为0；

var beats = ['ant', 'bison', 'camel', 'duck', 'bison'];
console.log(beats.indexOf('camel'));  // 2
console.log(beats.indexOf('camels')); // -1

2.20 Array.lastIndexOf()

lastIndexOf()方法返回指定元素在数组中的最后一个的索引，如果不存在则返回-1。从数组的后面向前查找，从fromIndex处开始。 语法：arr.lastIndexOf(searchElement, fromIndex); searchElement: 被查找的元素； fromIndex: 可选，从此位置开始逆向查找；

var beats = ['ant', 'bison', 'camel', 'duck', 'bison'];
console.log(beats.lastIndexOf('duck'));  // 3
console.log(beats.lastIndexOf('ducks')); // -1

遍历方法

2.21 Array.forEach()

forEach()方法对数组的每个元素执行一次提供的函数 语法: array.forEach((item, index, array) => {}, thisArg);

var arrs = ['a','b','c'];
arrs.forEach((item) => {
  console.log(item);
});
// a
// b
// c

2.22 Array.entries()

entries()方法返回一个新的Array iterator对象，该对象包含数组中每个索引的键/值对。 返回一个新的Array迭代器对象。 语法： array.entries();

var array1 = ['test','add','create', 'game'];
var iterator = array1.entries();
console.log(iterator.next()); // {value: [0, "test"], done: false}
console.log(iterator.next()); // {value: [1, "add"], done: false}
console.log(iterator.next()); // {value: [2, "create"], done: false}

2.23 Array.every()

every()方法返回一个布尔值，测试一个数组内的所有元素是否能通过某个指定函数的测试,如果每个元素都通过测试则返回true，若至少有一个不通过则返回false。 语法： array.every(callback, thisArg) 注意：若收到一个空数组，此方法在一切情况下都会返回true；

var array = [1,30,39,10,14];
var test = array.every((item, index, array) => {
  return item > 10;
});
console.log(test); // false

// 另外：测试一个空数组[]
console.log([].every((item, index, array) => {
  return item > 10;
})) // true;

2.24 Array.some()

some()方法测试数组中每个元素，若有一个元素通过测试则返回true，若都不通过则返回false； 语法：array.some(callBack, thisArg); 注意：如果用一个空数组进行测试，在任何情况下它返回的都是false。

var array = [11,23,39,14,89,57,64];
var even = (element) => {
  return element > 20
}
console.log(array.some(even)); // true;

// 另外：
var even0 = (element) => {
  return element > 100
}
console.log([].some(even)); // false;

2.25 Array.filter()

filter()创建一个新的数组，新数组中的元素是通过提供函数测试后返回的符合要求的数组； 语法：newArray = arr.filter(callback, thisArg);

var words = ['green', 'red', 'blue', 'yellow'];
var newArray = words.filter((item) => {
 return item.length > 4;
});
console.log(newArray); // ["green", "yellow"]

2.26 Array.find()

find()方法返回数组中满足提供的测试函数的第一个元素的值。否则返回undefined; 语法： array.find(callback, thisArg);

var array1 = [5, 12, 8, 30, 44];
var find = array1.find((item) => {
  return item > 10;
});
var find0 = array1.find((item) => {
  return item > 50;
});
console.log(find); // 12
console.log(find0); // undefined

2.27 Array.findIndex()

findIndex()方法返回数组中满足提供的测试函数的第一个元素的索引。否则返回-1；

var array1 = [5,12,8,19,44];
var indexFind = array1.findIndex((item) => {
  return item > 13;
});
var indexOther = array1.findIndex((item) => {
  return item > 53;
});
console.log(indexFind); // 3
console.log(indexOther); // -1

2.28 Array.keys()

keys()方法返回一个包含数组中每个索引键的Array Iterator对象 语法：array.keys();返回一个新的Array迭代器对象。

var array = ['a','b','c'];
var iterator = array.keys();
for(let key of iterator){
  console.log(key, iterator);
}
// 0 Array Iterator {}
// 1 Array Iterator {}
// 2 Array Iterator {}

// 索引迭代器会包含那些没有对应元素的索引
var arr = ['aa',,"c"];
var spKeys = Object.keys(arr);
var deKeys = [...arr.keys()];
console.log(spKeys); // ["0", "2"]
console.log(deKeys); // [0, 1, 2]

2.29 Array.map()

map方法创建一个新数组，其结果是该数组中的每个元素都调用一个提供的函数后返回的结果。 语法：array.map(callback, thisArg);

var array = [1,2,3,4,5];
var map = array.map(item => item * 3);
console.log(map); // [3, 6, 9, 12, 15]

2.30 Array.reduce()

reduce()方法对数组中的每个元素执行一个由您提供的reducer函数(升序执行)，将其结果汇总到单个返回值。 语法： array.reduce(callback(accumulator, currentValue, index, array), initalValue); accumulator: 累计值； currentValue：当前值； index：当前索引； array：数组； initalValue: 可选；作为累计数accumulator的初始值，如果没有则使用数组的第一个值作为初始值； 注意：如果没有提供initalValue，reduce会从索引1的地方开始执行callback方法，跳过第一个索引。如果提供initalValue，则从索引0开始。

var add = [2,3,4].reduce((acc,cur,index,array) => {
	// console.log(acc,cur,index,array);
	return acc + cur;
}, 30);
console.log(add); // 39

2.31 Array.reduceRight()

reduceRight()方法接受一个函数作为累加器，从右到左将数组的每个值进行累计处理；而reduce()方法是从左到右计算； 语法：Array.ruduceRight(callback(accumlator, currentValue, index, array), initalValue); 注意：如果没有提供intiValue，reduceRight会从索引最后第2个数开始，跳过第一个索引。如果提供initalValue,则从索引最后一个开始；

var add = [2,3,4].reduceRight((acc, cur, index, array) => {
  // console.log(acc, cur, index, array);
  return acc + cur;
}, 30)
console.log(add); // 39

2.32 Array.values()

values()方法返回一个新的Array Iterator对象，该对象包含数组每个索引的值； 语法：array.values(); 返回一个新的Array迭代对象

var arr = ['w','y','z'];
var eArr = arr.values();
for(var letter of eArr){
  console.log(letter);
}
// w
// y
// z

其它

2.33 Array.flat()

flat()方法会按照一个指定的深度递归遍历数组，并将所有元素与遍历到的子数组中的元素合并为一个新数组返回； 语法：array.flat(depth) depth: 可选，指定要提取嵌套数组的结构深度，默认值为1。

var arr1 = [1,2,[4,5]];
console.log(arr1.flat()); // [1,2,4,5]
var arr2 = [2,3,4,[5,[6,[7]]]];
console.log(arr2.flat(3)); // [2, 3, 4, 5, 6, 7]

2.34 Array.flatMap()

flatMap()方法首先使用映射函数映射每个元素，然后将结果压缩成一个新数组。它与map连着深度值为1的flat几乎相同，但是flatMap通常在合并成一种方法的效率稍微高一些。 语法：new_array = arr.flatMap(function(current, index, array){ // 返回新数组的元素； }, thisArg); 返回值为一个新的数组，其中每个元素都是回调函数的结果，并且结构深度depth值为1。

var arr1 = [2,3,4];
var newArr = arr1.map(val => [val*2]);
console.log(newArr); // [[4],[6],[8]]

var newArr1 = arr1.flatMap(x => x*2);
console.log(newArr1); // [4,6,8]

// 只会将flatMap中的函数返回的数组"平压"一层
var newArr2 = arr1.flatMap(x => [[x*2]]);
console.log(newArr2); // [[4],[6],[8]]

2.36 Array.apply()

apply()方法调用一个具有给定this值得函数，以及作为一个数组（或类似数组对象）提供的参数。 注意：call()方法的作用和apply()方法类似，区别就是call()方法接受的是参数列表，而apply()方法接受的是一个参数数组。 语法：fun.apply(thisArg, [argsArray]); thisArg: 可选。在func函数运行时使用的this的值。请注意，this可能不是该方法看到的实际值：如果这个函数处于非严格模式下，则指定为null或undefined时会自动替换为指向全局对象，原始值会被包装。 argsArray：可选的。一个数组或者类数组对象，其中的数组元素将作为单独的参数传给func函数。如果该参数的值为null或undefined，则表示不需要传入任何参数。

var numbers = [5,6,7,8,9];
var max = Math.max.apply(null, numbers);
console.log(max);// 9
var min = Math.min.apply(null, numbers);
console.log(min); // 5

var array = ['a','b'];
var elements = [1,2,3];
array.push.apply(array, elements);
console.log(array); // ["a", "b", 1, 2, 3]

2.38 Array.call()

call()方法使用一个指定的this值和单独给出的一个或多个参数来调用一个函数。 语法：fun.call(thisArg, arg1,arg2,arg3,…); thisArg: 在fun函数运行时指定的this值。 arg1，arg2，…: 指定的参数列表。

function Project(name, price){
  this.name = name;
  this.price = price;
}
function Food(name, price){
  Project.call(this, name, price);
  this.cate = 'food';
}
console.log(new Food('cheese', 5).name); // cheese

2.37 Array.bind()

bind()方法创建一个新的函数，在bind()被调用时，这个新函数的this被bind的第一个参数指定，其余的参数将作为新函数的参数供调用时使用。 语法：function.bind(thisArg, arg1, arg2, arg3…);
