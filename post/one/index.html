React Hooks的介绍

Oct 21, 2019
1 min read

常用hooks

1、useState

useState主要是用于定义和管理本地状态

import React, {useState} from 'react';
function Example(){
  const [count, setCount] = useState(0);
  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => {setCount(count + 1)}}>Click me</button>
    </div>
  )
}

useState参数是初始state，会返回一个数组。数组第一个值是state，第二个值是改变state的函数。 count和setCount是一一配对的，count的值只能通过setCount修改。

2、useEffect

useEffect的代码是在每次渲染后运行，包括第一次渲染； useEffect它跟类组件中的componentDidMount、componentDidUpdate和componentWillUnmount具有相同的用途，只不过被合并成了一个API。

2.1、使用useEffect完成componentDidMount的功能

import React, {useEffecr} from 'react';
function Example(){
  useEffect(() => {
    // TODO
  },[]);
  return (
    <p>Hello!</p>
  )
}

2.2、使用useEffect完成componentWillUpdate的效果

import React, {useEffect} from 'react';
function Example () {
  useEffect(() => {
    // TODO
  });
  return (<p>Hello hooks!</p>)
}

2.3、使用useEffect完成componentWillUnmount的效果

import React, {useEffect} from 'react';
function Example(source) {
  useEffect(() => {
    const subscribe = source.subscribe();
    // TODO
    return () => {
      subscribe.unsubscribe();
    }
  }, [source]); // 表示source改变时就执行一次
  return (
    <div>hello hooks!</div>
  )
}

3、useLayoutEffect

与useEffect一致，用于获取元素渲染结果；

4、useMemo,useCallback

与shouldComponentUpdate一致；

5、hooks使用规则

Hooks就是javascript函数，但是使用它们会有两个额外的规则：

5.1 只在函数最顶层调用Hooks，不要在循环、条件判断或者子函数中调用

5.2 只在函数组件和hooks自定义中调用hooks,不要在其它Javascript中调用

6、自定义hooks

6.1 自定义hook是一种自然遵循hook设计的约定，而并非React的特性

6.2 自定义hooks必须以use开头，这个约定非常重要
