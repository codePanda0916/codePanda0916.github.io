javascript引用类型--Array类型

Jan 20, 2020
3 min read

简介：引用类型的值（对象）是引用类型的一个实例。

所有对象都具有toLocaleString(),toString()和valueOf()方法

Array类型

1，创建数组的方式有两种

// 第一种：是使用Array构造函数
var colors = new Array();
var others = new Array(20, 'xyj');
// 第二种：是使用数组字面量表示法
var colors = [];
var others = [20, 'xyj'];

2，检测数组的方法

方法一：instanceof，仅用于只有一个全局执行环境

// 方法一：
if(value instanceof Array){
  // 对数组进行某些操作
}
// 方法二：Array.isArray(value)
if(Array.isArray(value)){
  // 对数组执行某些操作
}

3，转换方法

3.1，toLocaleString()
var colors = ['red', 'green', 'yellow'];
console.log(colors.toLocaleString());
// red,green,yellow
3.2， toString()
var colors = ['red', 'green', 'yellow'];
console.log(colors.toString());
// red,green,yellow
3.3， valueOf()
var colors = ['red', 'green', 'yellow'];
console.log(colors.valueOf());
// ["red", "green", "yellow"]

4，栈方法

栈是一种先进后出(Last-In-First-Out)的数据结构；push()和pop()来实现类似栈的行为。

var colors = [];
// push(),可以接收任意数量的参数，然后逐个添加到数组末尾。
colors.push('red', 'green')
console.log(colors); // ["red", "green"]
// pop()则从数组末尾移除最后一项，减少数组的length值；
colors.pop();
console.log(colors); // ["red"]

5，队列方法

队列数据结构的访问规则是FIFO(first-in-first-out)，先进先出， shift()方法是在数组前端移除第一项，unshift()方法是在数组的前端添加任意个项并返回新数组的长度。

如下：两种队列方式
// shift与push组成一个队列(前出后进)
var colors0 = ['red', 'green'];
colors0.push('yellow');
colors0.shift();
// pop与shift组成一个队列(前进后出)
var colors1 = ['red', 'green'];
colors1.unshift('yellow');
colors1.pop();

6，重排序方法

数组中可以直接用来重排序的方法有reverse()和sort();

6.1 reverse()
// reverse()反转数组项的顺序
var nums = [1,2,3,4,5,6,7];
nums.reverse();
nums; // [7, 6, 5, 4, 3, 2, 1]
6.2 sort()
// 由于reverse()不够灵活，因此才有了sort()方法
// sort()在默认情况下，按升序排列数组项，既最小的值位于最前面，最大的值位于最后面。
var arrs = [111,21,13,44,150,6,7];
arrs.sort();
arrs; // [111, 13, 150, 21, 44, 6, 7]

sort()函数接收一个比较函数作为参数，而比较函数则接收两个参数；

// Array.sort(compare(value1, value2));
// 如果value1 > value2，则返回正数
// 如果value1 = value2，则返回0
// 如果value1 < value2，则返回负数
// 如下为常用的比较函数compare；

// 1，升序排列
var arrs = [23,14,25,98,10,18];
var compare1 = function(value1, value2){
  return value1 - value2;
}
arrs.sort(compare1);
arrs; // [10, 14, 18, 23, 25, 98]

// 2，降序排列
var compare2 = function(value1, value2){
  return value2 - value1;
}
arrs.sort(compare2);
arrs; //  [98, 25, 23, 18, 14, 10]

7，操作方法

7.1，concat()，基于当前数组中的所有项创建一个新数组【不改变原数组】
var arrs = [1,2,3];
var copyArrs = arrs.concat(); // copyArrs复制了arrs数组。
var nextArrs = arrs.concat(44,55,14,23);
nextArrs; // [1, 2, 3, 44, 55, 14, 23]
7.2，slice()，能够基于当前数组中的一或多个项创建一个新数组。slice接收一个或两个参数【不改变原数组】
// 一个参数：从起始参数开始到结束；
var arrs = [1,2,3,4,5,6];
var newArrs1 = arrs.slice(2);
newArrs1; // [3,4,5,6]
// 两个参数：从起始参数开始到结束位置的前一个值；
newArrs2 = arrs.slice(2,3);
newArrs2; // [3];
7.3，splice()，主要用途是向数组的中部插入项【改变原数组】
// 1，删除：Array.splice(参数1，参数2)，参数1为要删除的第一项的位置，参数2为要删除的项数。

var arrs0 = [1,2,3,4,5,6,7,8];
arrs0.splice(3, 2);
arrs0; // [1, 2, 3, 6, 7, 8]

// 2，插入：Array.splice(参数1，0，参数2)，参数1为插入起始的位置，参数2为插入的项数。

var arrs1 = [1,2,3,4,5,6,7,8];
arrs1.splice(3,0,111,2222);
arrs1; // [1, 2, 3, 111, 2222, 4, 5, 6, 7, 8]

// 3，替换：Array.splice(参数1，n，参数2),参数1为替换起始的位置，n为要替换掉的(即要删掉的)项数，参数2为要插入替换的项数。

var arrs2 = [1,2,3,4,5,6,7,8];
arrs2.splice(3,2,111,2222);
arrs2; // [1, 2, 3, 111, 2222, 6, 7, 8]

8，位置方法

indexOf()和lastIndexOf()都接收两个参数：要查找的项和（可选的）表示查找起点位置的索引。如果没有找到则返回-1。

var numbers = [1,2,3,4,5,4,3,2,1];
console.log(numbers.indexOf(4));     // 3
console.log(numbers.lastIndexOf(4)); // 5
console.log(numbers.indexOf(4,4));   // 5

10，归并方法

ES5新增了两个归并数组的方法：reduce()和reduceRight()，这两个方法都接收两个参数–每一项上调用的函数和（可选的）作为归并基础的初始值。

函数接收4个参数：前一个值、当前值、项的索引和数组对象。

var arrs = [1,2,3,4,5];
var sum = arrs.reduce((pre, next, index ,array) => {
  return pre + next;
}, 100)
sum; // 115
