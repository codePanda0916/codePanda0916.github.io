javascript引用类型--String类型

Jan 22, 2020
2 min read

String类型是字符串的对象包装类型，可以像下面这样使用String构造函数来创建。

var stringObject = new String('hello world!');
stringObject; // {"hello world!"}
stringObject.length; // 12

String类型提供了很多方法；

1，字符串方法

1.1 charAt()

接收一个基于0的字符位置的参数，返回给定位置的那个字符。

var str = "hello world!";
str.charAt(4); // "o"
1.2 charCodeAt()

接收一个基于0的字符位置的参数，返回给定位置的那个字符的字符编码。

var str = "hello world!";
str.charCodeAt(0); // 104

2，字符串操作方法

2.1 concat()

用于将一或多个字符串拼接起来，返回拼接得到的新字符串。

var str = 'hello';
var res = str.concat('world');
console.log(str); // hello
console.log(res); // helloworld

但实际过程中，用到更多的还是操作符（+），比用concat方便多啦。

var str = "li";
var res = str + 'lei';
console.log(str); // li
console.log(res); // lilei
2.2 slice(startIndex, lastIndex)

startIndex：指定字符串的开始位置，lastIndex：指定的字符串最后一个字符串后面的位置，连个参数都可为负数。

var str = "good afternoon"
console.log(str.slice(2));   // "od afternoon"
console.log(str.slice(2,4)); // "od"
console.log(str.slice(-2));  // "on"
console.log(str.slice(-2, -1)); // "o"
2.3 substr(startIndex, n)

startIndex：指定字符串开始的位置，可为负数； n：字符串的个数，不可为负数，若为负数会自动转换为0；

var str = "abcdefgh";
console.log(str.substr(3));     // "defgh"
console.log(str.substr(-1,2));  // "h"
console.log(str.substr(-1,-1)); // ""
2.4 substring(startIndex, lastIndex)

startIndex,lastIndex都会把负数转换为0；

var str = "abcdefghi";
console.log(str.substring(2));    // "cdefghi"
console.log(str.substring(2,4));  // "cd"
console.log(str.substring(-2));   // "abcdefghi"
console.log(str.substring(2,-2)); // "ab"

3，字符串位置方法

3.1 indexOf()

从字符串的开头向后搜索子字符串，可接收两个参数，第一个参数是要搜索的子字符串，第二个参数是从字符串中的哪个位置开始搜索。搜不到返回-1；

var str = "hello world!";
console.log(str.indexOf('l'));   // 2
console.log(str.indexOf('l',4)); // 9
3.2 lastIndexOf()

从字符串的末尾向前搜索子字符串，可接收两个参数，第一个参数是要搜索的子字符串，第二个参数是从字符串中的哪个位置开始搜索。搜不到返回-1；

var str = "hello world!";
console.log(str.lastIndexOf('l'));   // 9
console.log(str.lastIndexOf('l',4)); // 3

4，trim()

这个方法创建一个字符串的副本，删除前置及后缀的所有空格，然后返回结果。

var str = "   good  ";
var trim = str.trim();
console.log(str);    // "   good  "
console.log(trim);   // "good"

5，字符串大小写转换方法

toLocaleLowerCase和toLocaleUpperCase是针对地区的方法，对于不知道自己代码将在哪种语言环境中运行的情况下，还是使用针对地区的方法更稳妥一些。

5.1 toLowerCase()

将字符串转换为小写

var str = "hello world!"
str.toLowerCase(); // "hello world!"
5.2 toLocaleLowerCase()

在针对地区的情况下，将字符串转换为小写

var str = "hello world!";
str.toLocaleLowerCase();  // "hello world!"
5.3 toUpperCase()

将字符串转换为大写

var str = "hello world!";
str.toUpperCase();  // "HELLO WORLD!"
5.4 toLocaleUpperCase()

在针对地区的情况下，将字符串转换为大写

var str = "hello world!";
str.toLocaleUpperCase();  // "HELLO WORLD!"

6，字符串的模式匹配方法

6.1 match()

接收一个参数，要么是一个字符串，要么是一个正则表达式或RegExp对象。本质上与调用RegExp的exec()方法相同。

var text = "cat,mat";
var matchs = text.match(/at/);
console.log(matchs); // ["at", index: 1, input: "cat,mat", groups: undefined]
6.2 search()

接收一个参数，这个参数要么是个字符串，要么是一个正则表达式或RegExp对象。返回第一项的索引。

var text = "cat,bat,sat,fat";
var pos = text.search(/at/);
console.log(pos); // 1
6.3 replace()

接收两个参数，第一个参数是一个字符串或者RegExp对象，第二个参数是一个字符串或者一个函数。

var text = "cat,bat,sat,fat";
var res = text.replace(/at/g, 'A');
var rest = text.replace('at', 'A');
console.log(text);  // "cat,bat,sat,fat"
console.log(res);   // "cA,bA,sA,fA"
console.log(rest);  // "cA,bat,sat,fat"
6.4 split()

可以基于指定的分隔符将一个字符串分割成多个子字符串，并将结果放在一个数组中，分隔符可以是字符串，也可以是一个RegExp对象，split()方法可以接受可选的第二个参数，用于指定数组的大小，以便确保返回的数组不会超过既定大小。

var text = "red,blue,green,yellow";
var color0 = text.split();    // ["red,blue,green,yellow"]
var color1 = text.split(','); // ["red", "blue", "green", "yellow"]
var color2 = text.split(',', 2); // ["red", "blue"]

7，localeCompare()

这个方法比较两个字符串，并返回下列值中的一个：

// 1, 如果字符串在字母表中应该排在字符串参数之前，则返回一个负数（大多数情况下是-1，具体的值要视实现而定）；
// 2，如果字符串等于字符串参数，则返回0；
// 3，如果字符串在字母表中应该排在字符串参数之后，则返回一个正数（大多数情况下是-1，具体的值要视实现而定）
var str = "yellow";
console.log(str.localeCompare('zoo'));   // -1
console.log(str.localeCompare('yellow'));// 0
console.log(str.localeCompare('brick')); // 1
// 另外大写字母是排在小写字母前面的

8，fromCharCode()

这个方法的任务是接收一或多个字符串编码，然后将它们转换成一个字符串。从本质上来看，这个方法与实例方法charCodeAt()执行的是相反的操作。

console.log(String.fromCharCode(104,101,108,108,111)); // hello
// 这里是给fromCharCode()传递的是字符串"hello"中每个字母的字符编码
